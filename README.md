# Rodando com Docker 🐳

Antes de rodar com docker, confira se as portas que serão utilizadas estão disponíveis, no caso dessa aplicação são utilizadas as seguintes portas: 3001 (backend), 3000 (frontend), 3306 (mysql).

```
docker-compose up -d
```
